### Instructions for Running the Server and Client

### Compiling the Code

#### For `server.c`:
```
gcc server.c -o server
```
#### For `client.c`:
```
gcc client.c -o client
```
### Running the executable file

#### For `server.c`:
```
./server
```
#### For `client.c`:
```
./client
```
